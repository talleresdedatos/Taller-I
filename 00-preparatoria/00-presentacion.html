Instalacin de R

Instalar desde The Comprehensive R Archive Network - CRAN.

  • Opcin 1: Googlear “Descargar R”
  • Opcin 2: Ir directamente al CRAN

Elegir la versin (mnimo 4.1) adecuada para el sistema operativo a utilizar y elegir todas las opciones por defecto una vez iniciado el instalador, es decir, no cambiar ninguna opcin.

Si ya tuvieras una versin de R previa en la que trabajas cotidianamente, hzmelo saber para guiarte en la actualizacin de paquetes.

Instalacin de R Studio

Instalar R Studio Desktop desde la web de RStudio

Elegir la versin (mnimo v 1.4) adecuada para el sistema operativo a utilizar y elegir todas las opciones por defecto una vez iniciado el instalador, es decir, no cambiar ninguna opcin.

Instalacin de paquetes

En la consola de R dentro de RStudio, utilizar el siguiente cdigo lnea por lnea:

install.packages(c("tidyverse", "rmarkdown", "rsconnect", "remotes"))
remotes::install_github("calderonsamuel/templatesRmd")

Cuenta en Rpubs.com

Rpubs.com es una plataforma en la que puedes subir gratuitamente trabajos realizados con R Markdown desde RStudio. Es usado por muchas instituciones en sus cursos de Estadstica, Machine Learning, Ciencia de Datos, etc.

  1. En un navegador, dirgete a https://rpubs.com/.
  2. Haz click en “Register” para crear una cuenta nueva.
  3. Llena el formulario de registro. Recomiendo usar un username sobrio/profesional.

Para usar la plataforma en el futuro bastar con ingresar (Sign in).

Prueba de Knit

Crear un documento R Markdown dentro de RStudio.

  • Debes ir File > New file > R Markdown
  • Selecciona “From template” y elige “Plantilla ejemplo” del paquete templatesRmd.
  • Cambia los parmetros author y date en el encabezado del documento.

Renderiza el documento con el botn Knit.

Subir el contenido a Rpubs.com

En el documento renderizado, ubica el botn “Publish” en la esquina superior derecha. Elige la opcin Rpubs.com y llena los detalles solicitados por el navegador.
